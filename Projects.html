Projects using Firm

Frontends

  • CParser: GCC-compatible C frontend with full C99 support

  • X10i: X10 native code compiler (for invasic architectures)

  • bytecode2firm: Java bytecode frontend

  • Fluffy: Frontend for the fluffy language

  • brainfuck: Brainfuck frontend (and a good tutorial)

Tools

  • yComp: Visualization tool for intermediate representation graphs

  • Sisyphus: A generic regression tester aimed at compiler tests

Libraries

  • jFirm: Java bindings for libFirm

  • libOO: Support library for dealing with object-oriented constructs

Testing

  • firm-testsuite: C compiler and Firm optimization testsuite
