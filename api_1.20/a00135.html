Detailed Description

ir_node - a datatype representing a Firm node

The common fields are:

  • arity - The number of predecessors in the Firm graph.
  • in - A list with the predecessors in the Firm graph. There are routines to access individual elements and to obtain the array. The method returning the array should not be used.
  • mode - The mode of the node. There are routines to get the mode but also to access the mode's fields directly.
  • opcode - The opcode of the node. There are routines to get the opcode but also to access the opcode's fields directly.
  • node_nr - A unique number for the node. Available only if debugging is turned on.
