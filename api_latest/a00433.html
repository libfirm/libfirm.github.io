ir_op * op_Mod
 Mod opcode.
 

Detailed Description

returns the remainder of its operands from an implied division.

Examples:

mod(5,3) produces 2 mod(5,-3) produces 2 mod(-5,3) produces -2 mod(-5,-3) produces -2
